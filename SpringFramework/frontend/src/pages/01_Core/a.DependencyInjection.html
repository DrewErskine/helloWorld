Dependency Injection

Dependency Injection (DI) is a design pattern used to implement IoC (Inversion of Control), allowing the creation of dependent objects outside of a class and providing those objects to the class. In Spring, DI is a core feature, making the application loosely coupled, easier to test, and more maintainable.

Types of Dependency Injection

Constructor Injection

@Component
public class ExampleService {
    private final ExampleRepository repository;

    @Autowired
    public ExampleService(ExampleRepository repository) {
        this.repository = repository;
    }
}

Setter Injection

@Component
public class ExampleService {
    private ExampleRepository repository;

    @Autowired
    public void setRepository(ExampleRepository repository) {
        this.repository = repository;
    }
}

Field Injection

@Component
public class ExampleService {
    @Autowired
    private ExampleRepository repository;
}
