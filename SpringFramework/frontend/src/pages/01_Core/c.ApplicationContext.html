Application Context

The Application Context is the central interface to the Spring IoC container. It is responsible for instantiating, configuring, and assembling beans.

Types of ApplicationContext

  • ClassPathXmlApplicationContext: Loads context definition from an XML file in the classpath.
  • FileSystemXmlApplicationContext: Loads context definition from an XML file in the file system.
  • AnnotationConfigApplicationContext: Loads context from annotated classes and Java-based configuration.

Example

ApplicationContext context = new AnnotationConfigApplicationContext(AppConfig.class);
ExampleService service = context.getBean(ExampleService.class);
