Expression Language (SpEL)

Spring Expression Language (SpEL) is a powerful language that supports querying and manipulating an object graph at runtime. It is similar to other expression languages but integrates seamlessly with Spring.

Example

@Value("#{T(java.lang.Math).random() * 100.0}")
private double randomNumber;

@Value("#{exampleService.someProperty}")
private String someProperty;

Usage

Annotations

@Component
public class ExampleComponent {
    @Value("#{exampleBean.someProperty}")
    private String someProperty;
}

XML Configuration

<bean id="exampleBean" class="com.example.ExampleBean">
    <property name="someProperty" value="#{anotherBean.someOtherProperty}" />
</bean>
